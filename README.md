# RENTAL-X-MEDIA-QUERY-PR

#<h1>🏔️ RENTAL-X – Mountain Cabin & Adventure Website</h1>

RENTAL-X is a modern, visually rich **travel & cabin rental website** built using **HTML5 and CSS3**.  
The design is inspired by nature and adventure, perfect for showcasing mountain cabins, hiking tours, and cycling trips.

## ✨ Features

### 🌄 Hero Banner
- Full-screen banner image
- Inspirational tagline overlay
- Dark overlay for better readability

### 🧭 Navigation Bar
- Transparent navbar over banner
- Logo + menu links (Home, About, Blog, Resources)
- Clean & minimal UI

### 🏕️ Mountain Cabin Section
- 4 featured cabins
- Image cards with hover zoom & brightness effect
- Title and description overlay on images

### 🥾 Adventure Sections
- **Hiking Tours**
- **Cycling Group Tours**
- Two-column layout with image + content
- Call-to-action buttons

### 🦶 Footer
- Product & category links
- Company information
- Social media icons (Instagram, Facebook, YouTube)
- Fully responsive footer layout
